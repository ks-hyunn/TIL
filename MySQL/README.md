# 데이터베이스의 정의와 특징

---

### 데이터베이스

- '데이터의 집합'

- 여러 명의 사용자나 응용프로그램이 공유하는 데이터들

- 동시에 접근 가능해야

- 데이터의 저장 공간 자체

### DBMS

- 데이터베이스를 관리, 운영하는 역할

---

### 관계형 DBMS의 장단점

- 장점
  - 다른 DBMS에 비해 업무가 변화될 경우 쉽게 변화에 순응
  - 유지보수 측면에서도 편리
  - 대용량 데이터의 관리와 데이터 무결성(Integration)보장
- 단점
  - 시스템 자우언을 많이 차지해 시스템이 전반적으로 느려지는것
    - 하드웨어 발전되어 해결
